Home

This is the Home Page for the ASDL IoT Development Project. This team is developing a series of sensors for use across multiple applications to connect Georgia Tech's campus, and more. Real-time sensor data can be found from any of the online sensors in the Sensor List. If you want to learn more about how this project works, visit the About Us tab.

This webpage, the hardware that collects its data, and its backend data processing is the work of Alex Lomis (2022-2023).

Sensor List

A list of registered sensors and their rough purpose can be found below. Next to each sensor is a point showing the last time (in GMT - 0500) that the sensor uploaded data to the cloud. This is a good metric of if the sensor is online and functioning. Click on a sensor's name to view more specifics about a sensor, as well as its data.
